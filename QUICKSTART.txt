🐛 TROUBLESHOOTING
───────────────────────────────────────────────────────────────────────────────

Issue: "Cannot connect to database"
  → Make sure MongoDB is running
  → Check .env file has correct MONGODB_URI
  → Verify database server is accessible

Issue: "Images not loading"
  → Check if server is running (it serves images)
  → Verify /uploads folder exists
  → Try refreshing the page

Issue: "Login fails"
  → Use exact credentials (case-sensitive)
  → For shop login, use "Kiran Dairy Kharadi" exactly
  → Check browser console for errors

Issue: "Inventory shows 0"
  → Run npm run seed:demo again to reset
  → Verify stock wasn't sold out
  → Check browser cache (Ctrl+Shift+Delete)

Issue: "POS sale failed"
  → Check quantity doesn't exceed available stock
  → Verify payment method selected
  → Check browser console for detailed error

───────────────────────────────────────────────────────────────────────────────

   Secure token-based login
   7-day session validity

✅ Role-Based Access
   Admins: Full system access
   Shops: Only see their data

✅ Data Isolation
   Shop A cannot see Shop B's inventory
